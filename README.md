# AssistVoice ML Project

Проект для реализации машинного обучения на Python с нуля.

## Структура проекта

- `data/` — данные
- `notebooks/` — Jupyter ноутбуки
- `src/` — исходный код
  - `data/`, `features/`, `models/`, `train/`, `predict/`, `utils/`
- `tests/` — автоматические тесты
- `scripts/` — утилитные скрипты
- `requirements.txt` — зависимости
